## Шаг 4 (Редактор ex, команды)
1) Для того, чтобы сохранить файл с новым именем, была использована команда: `:w letter_new.txt`

2) Для подсчета количества символов слов, строк в текущем файле, была использована команда `:!wc -mwl %`:
```bash
[Изменения не сохранены] 35 136 1512 letter.txt
```

То есть получаем:
  - 35 - строк
  - 136 - слов
  - 1512 - символов
  - letter.txt - имя текущего файла

Команда:
  - :! - выполнить команду терминала
  - wc - подсчёт
  - m - символы
  - w - слова
  - l - строки
  - % - текущий файл

3) Для выхода из редактора без сохранения отредактированного файла, была использована команда `:q!`:
  - :q - выйти
  - ! - принудительно, даже если есть несохранённые изменения

4) Для вставки содержимого /etc/hostname в letter.txt, была использована команда `:r /etc/hostname`.

Команда:
  - r - прочитать из файла в текущий текст, то есть фактически вставить
  - /etc/hostname - путь к файлу, содержимое которого нужно вставить

## Шаг 5 (Редактор ex, поиск и замена)
1) Для поиска по произвольному образцу в редакторе vi, были выполнены команды:
  - `/образец` - поиск вперёд (Например, /Ассоциация)
  - `?образец` - поиск назад (Например, ?Ассоциация)

Переход между найденными совпадениями:
  - `n` - следующее совпадение
  - `N` - предыдущее совпадение

2.1) Для замены первого пробела на подчеркивание в текущей строке, была использована команда `:s/ /_/`.

Команда:
  - s - команда замены
  - первый `/` - отделяет команду от шаблона поиска
  - пробел между `/ /` - символ, который нужно найти
  - второй `/` - отделяет искомый символ от замены
  - `_` - символ, на который заменяется пробел
  - последний `/` — завершает команду

2.2) Для замены всех пробелов на подчеркивание в текущей строке, была использована команда `:s/ /_/g`.

Команда:
  - s - команда замены
  - первый `/` - отделяет команду от шаблона поиска
  - пробел между `/ /` - символ, который нужно найти
  - второй `/` - отделяет искомый символ от замены
  - `_` - символ, на который заменяется пробел
  - третий `/` - завершает основную часть команды замены
  - g - указывает выполнить замену всех пробелов в текущей строке

2.3) Для замены всех пробелов на подчеркивание в строках с 5 по 10, была использована команда `:5,10s/ /_/g`.

Команда:
  - `5,10` - диапазон строк, в которых будет выполняться команда, то есть с 5-й по 10-ю строку включительно
  - s - команда замены
  - первый `/` - отделяет команду от шаблона поиска
  - пробел между `/ /` - символ, который нужно найти
  - второй `/` - отделяет искомый символ от замены
  - `_` - символ, на который заменяется пробел
  - третий `/` - завершает основную часть команды замены
  - g - указывает заменить все пробелы в каждой строке указанного диапазона

2.4) Для замены всех пробелов на подчеркивание во всем тексте, была использована команда `:%s/ /_/g`.

Команда:
  - `%` - обозначает весь текст файла, то есть все строки
  - s - команда замены (substitute)
  - первый `/` - отделяет команду от шаблона поиска
  - пробел между `/ /` - символ, который нужно найти
  - второй `/` - отделяет искомый символ от замены
  - `_` - символ, на который заменяется пробел
  - третий `/` - завершает основную часть команды замены
  - g - указывает заменить все пробелы в каждой строке файла
