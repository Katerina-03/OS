## Шаг 2
* Для открытия файла `letter.txt` была введена команда `vi letter.txt`.

* Для перемещения курсора без использования стрелок применялись клавиши:
   - `h` - влево
   - `l` - вправо
   - `k` - вверх
   - `j` - вниз

* Навигация по словам осуществлялась так:
   - `w` - перейти к началу следующего слова
   - `b` - перейти к началу предыдущего слова
   - `e` - перейти в конец текущего или следующего слова

* Перемещение по строкам выполнялось с помощью команд:
   - `0` - в начало текущей строки
   - `$` - в конец текущей строки
   - `gg` - перейти к первой строке документа
   - `G` - перейти к последней строке документа

## Шаг 3
* Чтобы добавить текст и выйти из режима вставки, применялись:
   - `i` - переход в режим вставки
   - `Esc` - возврат в командный режим

* Удаление символа под курсором - командой `x`. Вставка удалённого символа после курсора - `p`. Также можно использовать `P` для вставки перед курсором.

* Удаление слова, на котором находится курсор, - `dw`. Вставка удалённого слова после курсора - `p`.

* Удаление текущей строки - `dd`. Вставка удалённой строки после курсора - `p`.

* Удаление нескольких строк (например, `ndd`, где `n` - количество строк). Вставка удалённых строк после курсора - `p`.

* Команды перемещения с числовым повтором:
   - `10h` - на 10 символов влево
   - `10l` - на 10 символов вправо
   - `20j` - на 20 строк вниз
   - `20k` - на 20 строк вверх
   - `5w` - на 5 слов вперёд
   - `3b` - на 3 слова назад

* Копирование текущего символа в буфер - `yl`. Вставка скопированного символа после курсора - `p`.

* Копирование текущего слова - `yw`. Вставка - `p`.

* Копирование текущей строки - `yy`. Вставка - `p`.

* Копирование группы строк - `nyy` (где `n` - количество строк). Вставка скопированных строк - `p`.

* Удаление трёх символов - `3x`.

* Удаление четырёх слов - `4dw`.

* Удаление пяти строк - `5dd`.

## Шаг 4

* Чтобы сохранить редактируемый файл под новым именем, применялась команда:  
   `:w letter_new.txt`

* Для подсчёта количества строк, слов и символов в текущем файле использовалась команда:  
   `:!wc -mwl %`

   Что означают части команды:  
   - `:!` - выполнить внешнюю команду терминала  
   - `wc` - утилита подсчёта (word count)  
   - `-m` - считать символы  
   - `-w` - считать слова  
   - `-l` - считать строки  
   - `%` - путь к текущему открытому файлу

* Для выхода из редактора без сохранения изменений использовалась команда:  
   `:q!`

   Расшифровка:  
   - `:q` - выход (quit)  
   - `!` - принудительное выполнение, игнорируя несохранённые изменения

* Чтобы вставить содержимое файла `/etc/hostname` в текущую позицию курсора в `letter.txt`, применялась команда:  
   `:r /etc/hostname`

   Пояснение:  
   - `:r` - прочитать (read) содержимое указанного файла и вставить его в текущий документ  
   - `/etc/hostname` - путь к файлу, содержимое которого вставляется

## Шаг 5 

* Для поиска по произвольному образцу в редакторе vi используются команды `/слово` для поиска вперёд и `?слово` для поиска назад. Перемещение между найденными совпадениями осуществляется клавишами `n` для перехода к следующему совпадению и `N` для перехода к предыдущему.

* Для выполнения замены символов используется команда `s` (substitute). Чтобы заменить первый пробел на подчёркивание в текущей строке, применяется команда `:s/ /_/`, где первый слеш отделяет команду от шаблона поиска, пробел является искомым символом, второй слеш отделяет искомый символ от символа замены, подчёркивание - это то, на что заменяется пробел, а последний слеш завершает команду.

* Если необходимо заменить все пробелы на подчёркивание в пределах текущей строки, используется команда `:s/ /_/g`, где добавленный флаг `g` указывает на необходимость замены всех вхождений в строке.

* Для замены всех пробелов на подчёркивание в диапазоне строк, например с 5-й по 10-ю включительно, применяется команда `:5,10s/ /_/g`, где `5,10` задаёт диапазон строк. Флаг `g` обеспечивает замену всех пробелов в каждой строке указанного диапазона.

* Наконец, чтобы заменить все пробелы на подчёркивание во всём тексте файла, используется команда `:%s/ /_/g`, где символ `%` обозначает весь файл, то есть все строки. Остальные части команды работают так же, как описано выше, а флаг `g` указывает на замену всех пробелов в каждой строке файла.
