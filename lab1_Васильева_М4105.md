**Шаг 3. Результат выполнения команд:**

Cписок имен пользователей, работающих в системе, и их общее количество
`echo "who -q" > commands`

Текущая дата в формате: день месяц_полностью год
`echo 'date "+%d %B %Y"' >> commands`

Календарь за 1752 год в файл calendar
`echo 'cal 1752 > calendar' >> commands`

Вывести calendar с номерами непустых строк
`echo 'cat -b calendar' >> commands`

Подсчет слов в calendar
`echo 'wc -w calendar' >> commands`

Код возврата 0 указывает на корректное выполнение команды
