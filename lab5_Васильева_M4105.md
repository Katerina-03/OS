1. Вывести строки из двух символов, не являющихся цифрами

`grep -x '[^0-9][^0-9]'` - `-x` требует совпадения со всей строкой, `[^0-9]` - любой символ кроме цифр
 
```bash
ekate@DESKTOP-SMCGLNG:~$ grep -x '[^0-9][^0-9]' text.txt
ab
xy
```

2. Вывести слова из одной буквы и номера строк

`grep -n -o '\b[a-zA-Z]\b'` - `-n` выводит номер строки, `-o` выводит только совпадение, `\b` - границы слова

```bash
ekate@DESKTOP-SMCGLNG:~$ grep -n -o '\b[a-zA-Z]\b' text.txt
3:b
4:c
31:A
31:B
31:C
32:a
32:a
32:a
36:a
36:b
36:c
```

3. Вывести слова, начинающиеся на “ab” и заканчивающиеся на “z”

`grep -o '\bab[a-zA-Z]*z\b'` - слова начинающиеся на ab и заканчивающиеся на z

```bash
ekate@DESKTOP-SMCGLNG:~$ grep -o '\bab[a-zA-Z]*z\b' text.txt
abz
abaaz
abz
abcz
abzabz
```

4. Вывести строки с трехзначным числом-палиндромом

`grep -E '\b([0-9])[0-9]\1\b` - трёхзначный палиндром (первая и третья цифры одинаковы)

```bash
ekate@DESKTOP-SMCGLNG:~$ grep -E '\b([0-9])[0-9]\1\b' text.txt
```

5. Вывести строки с двумя одинаковыми словами подряд

`grep -i -E '\b([a-zA-Z]+)\s+\1\b'` - `-i` игнорирует регистр, `\1` ссылается на первую группу

```bash
ekate@DESKTOP-SMCGLNG:~$ grep -i -E '\b([a-zA-Z]+)\s+\1\b' text.txt
hello world world
test test again
word Word word
hello Hello HELLO
a a a
```

6. Работа с системными файлами (.profile)

`grep -c '\.profile' ~/.profile 2>/dev/null || echo "0"`:

+ `grep -c` - подсчитывает количество строк с текстом .profile

+ `'\.profile'` - ищет именно точку, а не любой символ

+ `~/.profile` - файл в домашней папке

+ `2>/dev/null` - скрывает ошибки (если нет файла)

+ `|| echo "0"` - если ошибка, вывести 0

```bash
ekate@DESKTOP-SMCGLNG:~$ grep -c '\.profile' ~/.profile 2>/dev/null || echo "0"
1
```

Запись команд в файл commands

```bash
ekate@DESKTOP-SMCGLNG:~$ echo "grep -x '[^0-9][^0-9]' text.txt" > commands
ekate@DESKTOP-SMCGLNG:~$ echo "grep -n -o '\b[a-zA-Z]\b' text.txt" >> commands
ekate@DESKTOP-SMCGLNG:~$ echo "grep -o '\bab[a-zA-Z]*z\b' text.txt" >> commands
ekate@DESKTOP-SMCGLNG:~$ echo "grep -E '\b([0-9])[0-9]\1\b' text.txt" >> commands
ekate@DESKTOP-SMCGLNG:~$ echo "grep -i -E '\b([a-zA-Z]+)\s+\1\b' text.txt" >> commands
ekate@DESKTOP-SMCGLNG:~$ echo "grep -c '\.profile' ~/.profile 2>/dev/null || echo '0'" >> commands
ekate@DESKTOP-SMCGLNG:~$ echo "ls -l /usr/bin | grep '^...s'" >> commands
```
