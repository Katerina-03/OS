## Шаг 1 (справка)
1) **read** - читает данные со стандартного ввода и записывает их в переменные shell. Обычно используют, чтобы запросить ввод у пользователя в сценарии

2) **decare** - задаёт свойства переменной
   Флан **i** - объявляет переменную целочисленной 

3) **test** - встроенная команда для проверки условий: строковых, числовых и файловых. Возвращает код завершения, который потом используют if, while и другие конструкции

4) **[** - короткая форма команды test. По смыслу эквивалентна test, но пишется так: [ условие ]

5) **if** - оператор ветвления, где после if стоит не “булево выражение”, а список команд. Если последняя команда вернула код 0, ветка считается истинной и выполняется then. Сам if сравнения не делает - для этого обычно используют test или [

6) **select** - конструкция для простого текстового меню. Пользователю показывается нумерованный список вариантов, после чего он выбирает пункт по номер 

7) **while** - цикл с предусловием. Команды внутри цикла выполняются, пока последняя команда в условии возвращает код 0

8) **for** - цикл перебора. Переменная по очереди принимает значения из списка слов; если список не указан, берутся позиционные параметры сценария

2.4) **С помощью declare определите переменную int_var целого типа. Введите значение переменной с помощью команды read. Какое значение будет записано в переменную int_var если ввести: целое число; строку; вещественное число (например 3.14)?**
```bash

declare -i int_var

read int_var

# Выводим результат
echo "int_var = $int_var"
```

 - **Введем целое число**:
```bash

Введите значение int_var:
10
int_var = 10
```

То есть если ввести целое число, то в int_var запишется это же целое число.

 - **Введем строку**:
```bash
nozdryakov@debian:~$ ./int_var.sh
Введите значение int_var:

```

То есть при вводе строки, например java, shell интерпретирует её как имя переменной. Если такая переменная не определена, в int_var записывается 0.

То есть если ввести вещественное число, то получаем ошибку арифметического синтаксиса, так как такое значение не является целым числом.

Входит ли имя запускаемого скрипта в списки ${*}, ${@}? Сравните вычисление количества параметров с помощью ${#} и аналогичного параметра argc фукнции main языка Си (для 1 курса)**

## Шаг 4 (управляющие конструкции)
4.1) **Используя оператор if и команду test напишите сценарий root_test.sh, принимающий в качестве параметра командной строки имя предполагаемого пользователя. Если параметр не задан, то вывести "Отсутствует имя". Если параметр введен и совпадает со строкой "root", то вывести "Вы - суперпользователь", иначе вывести "Вы - обычный пользователь"**

```bash
nozdryakov@debian:~$ cat root_test.sh
#!/bin/sh
# Автор: Ноздряков Богдан M4106
# Назначение: проверка имени пользователя, переданного в командной строке
# Дата создания: 2026-03-26

# Если параметр не задан, выводим сообщение об отсутствии имени
if test -z "$1"
then
    echo "Отсутствует имя"
# Если параметр равен root, считаем пользователя суперпользователем
elif test "$1" = "root"
then
    echo "Вы - суперпользователь"
# Во всех остальных случаях это обычный пользователь
else
    echo "Вы - обычный пользователь"
fi
```

```bash
nozdryakov@debian:~$ chmod +x root_test.sh
nozdryakov@debian:~$ ./root_test.sh
Отсутствует имя
nozdryakov@debian:~$ ./root_test.sh root
Вы - суперпользователь
nozdryakov@debian:~$ ./root_test.sh bogdan
Вы - обычный пользователь
```

4.2) **Используя оператор select напишите сценарий poll_menu.sh организующий меню для голосования за одну из партий: ЯГОДА, КЛМН, МЕДРО, ВВЖ, СССР, ПДД**

```bash
nozdryakov@debian:~$ cat poll_menu.sh
#!/bin/bash
# Автор: Ноздряков Богдан M4106
# Назначение: меню голосования за одну из партий
# Дата создания: 2026-03-26

# Выводим приглашение для выбора пункта меню
echo "Выберите партию для голосования:"

# Организуем меню с помощью select
select party in "ЯГОДА" "КЛМН" "МЕДРО" "ВВЖ" "СССР" "ПДД"
do
    # Проверяем, был ли выбран корректный пункт меню
    if test -n "$party"
    then
        echo "Вы выбрали: $party"
        break
    else
        echo "Неверный выбор"
    fi
done
```

```bash
nozdryakov@debian:~$ chmod +x poll_menu.sh

1) ЯГОДА
2) КЛМН
3) МЕДРО
4) ВВЖ
5) СССР
6) ПДД
#? 3
Вы выбрали: МЕДРО
```
